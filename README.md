# DevRoast

O lugar onde seu código vai ser julgado — com humor.

## O que é?

DevRoast é uma plataforma que analiza seu código e te dá uma nota de roast. Pode ser brutalhamente honesto ou no modo "full roast" com máximo sarcasmo.

## Como usar

1. Cole seu código no editor
2. Escolha a linguagem
3. Ative o modo roast para sarcasm máximo
4. Clique em `$ roast_my_code`
5. Receba sua nota e vergonha pública

## Features

- **Análise de código** - Identifica anti-patterns, más práticas e código perigoso
- **Modo Roast** - Críticas com humor implacável
- **Modo Honest** - Feedback construtivo (mais ou menos)
- **Leaderboard** - Veja os códigos mais vergonhosos da internet
- **Syntax Highlighting** - Código destacado com tema bonito

## Exemplos de Roast

- "usando var? really? anos 2015 chiamando"
- "eval() para input de usuário? isso é um presentinho para hackers"
- "sem validation? ta convidando bugs pro café"
